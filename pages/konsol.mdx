# Konsol

Konsolga ma'lumotni chiqarish uchun console JavaScript obyektining bir nechta usuli mavjud bo'lib, ular bilan console.dir(console) tanishimiz mumkin. Ko'rsatiladigan usullar odatda quyidagilar:

- assert
- clear
- count
- debug
- dir
- dirxml
- error
- group
- groupCollapsed
- groupEnd
- info
- log
- profile
- profileEnd
- table
- time
- timeEnd
- timeStamp
- trace
- warn
