# JavaScript O'zgaruvchilari

|                     |                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------ |
| **ozgaruvchi_nomi** | **\{Majburiy\} O'zgaruvchi nomi: unga murojaat qilishda ishlatiladi.**                     |
| =                   | **[Ixtiyoriy]** Tayinlash (o'zgaruvchi qiymatini aniqlash)                                 |
| qiymat              | **\{Tayinlash ishlatilganda Majburiy\}** O'zgaruvchining qiymati **[standart: undefined]** |

O'zgaruvchilar JavaScriptning ko'p qismini tashkil qiladi. Bu o'zgaruvchilar raqamlardan tortib obyektlargacha bo'lgan narsalarni tashkil qiladi, ular butun JavaScript bo'ylab hayotimizni ancha osonlashtiradi.

## O'zgaruvchini aniqlash

```js
var myVariable = "Mana bu o'zgaruvchi!";
```
