## console.log() ni ishlatish

**Kirish**

Barcha zamonaviy web-brauzerlar, shuningdek, Node.js va deyarli barcha JavaScript muhitlari qayd qilish metodlari to'plamidan foydalanib konsolga xabar yozishni qo'llab-quvvatlaydi. Ushbu metodlardan eng keng tarqalgani `console.log(){:jsx}`.

Brauzer muhitida `console.log(){:jsx}` funksiyasi asosan nosozliklarni tuzatish uchun ishlatiladi.

**Ilk qadam**

Brauzeringizda JavaScript konsolini oching, quyidagilarni kiriting va `Enter` tugmasini bosing:

```js
console.log("Hello, World!");
```

Bu konsolga quyidagicha qayd qiladi:

![1-rasm](https://i.stack.imgur.com/Io1Iv.png)

Yuqoridagi misolda `console.log(){:jsx}` funksiyasi konsolga `"Hello, World!"{:jsx}` ni chop etadi va `undefined{:jsx}` qaytaradi (yuqoridagi konsolning chiqish oynasida ko'rsatilgan). Buning sababi, `console.log(){:jsx}` ning aniq qaytish qiymati yo'q.

**O'zgaruvchilarni chiqarish**

`console.log(){:jsx}` nafaqat stringlar, har qanday turdagi o'zgaruvchilarni qayd qilishi mumkin. Konsolda ko'rsatish uchun o'zgaruvchini kiriting, masalan:

![2-rasm](https://i.stack.imgur.com/oSX2F.png)

Agar siz ikki yoki undan ortiq qiymatlarni qayd qilmoqchi bo'lsangiz, ularni vergul bilan ajrating. Birlashtirish paytida har bir argument orasiga avtomatik ravishda bo'shliqlar qo'shiladi:

```js
var thisVar = "first value";
var thatVar = "second value";
console.log("thisVar:", thisVar, "and thatVar:", thatVar);
```

Bu konsolga quyidagicha qayd qiladi:

![3-rasm](https://i.stack.imgur.com/LuMaA.png)

**Joy egalari**

Siz console.log() dan joy egalari bilan birgalikda foydalanishingiz mumkin:

```js
var greet = "Hello",
  who = "World";
console.log("%s, %s!", greet, who);
```

Bu konsolga quyidagicha qayd qiladi:

![4-rasm](https://i.stack.imgur.com/KfHut.png)

**Obyektlarni qayd qilish**

Quyida biz obyektni qayd etish natijasini ko'ramiz. Bu ko'pincha API so'rovlaridan kelgan JSON javoblarini qayd qilishda foydali.

```js
console.log({
  Email: "",
  Groups: {},
  Id: 33,
  IsHiddenInUI: false,
  IsSiteAdmin: false,
  LoginName: "i:0#.w|virtualdomain\\user2",
  PrincipalType: 1,
  Title: "user2",
});
```
